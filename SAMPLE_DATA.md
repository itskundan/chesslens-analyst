# Sample Data for Testing

## CSV Format Example

Create a file called `sample_game.csv`:

```csv
White,Black
e4,e5
Nf3,Nc6
Bb5,a6
Ba4,Nf6
O-O,Be7
Re1,b5
Bb3,d6
c3,O-O
h3,Na5
Bc2,c5
d4,Qc7
```

This represents the opening moves of the Ruy Lopez (Spanish Opening).

## Image OCR Testing

For testing the OCR functionality, create an image (PNG or JPG) containing text like:

```
1. e4 e5
2. Nf3 Nc6
3. Bb5 a6
4. Ba4 Nf6
5. O-O Be7
6. Re1 b5
7. Bb3 d6
```

Tips for best OCR results:
- Use clear, high-contrast images
- Ensure text is horizontally aligned
- Use standard algebraic notation
- Avoid handwritten notation (typed is better)
- Good lighting with no glare or shadows
